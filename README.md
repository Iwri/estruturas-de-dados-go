# Estruturas de Dados em Go

Implementações de estruturas fundamentais estudadas na disciplina de Algoritmos e Estruturas de Dados 1 (UFRN) - DCA3503 .

## Conteúdo

* ArrayList (array dinâmico)
* Linked List (simples e duplamente encadeada)
* Stack (pilha)
* Queue (fila)
* Deque (fila de duas pontas)
* Algoritmos de busca:
  * Busca Linear
  * Busca Binária (iterativa e recursiva)

## Objetivo

Praticar e consolidar conceitos fundamentais de estruturas de dados através de implementações em Go.

## Linguagem

Go (Golang)
